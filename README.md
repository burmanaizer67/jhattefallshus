# JH Attefallshus - Conversionförbättringar

Det här repot innehåller en förbättrad startsida samt en drop-in patch för Smartproduktion.

## Innehåll

- [index.html](index.html)
  - Fristående, ny konverteringsoptimerad startsida.
- [smartproduktion-patch.html](smartproduktion-patch.html)
  - CSS + JS patch som appliceras på befintlig Smartproduktion-sida.
- [IMPLEMENTATION-SMARTPRODUKTION.md](IMPLEMENTATION-SMARTPRODUKTION.md)
  - Steg-för-steg guide för införande i produktion.
- [bonus-templates.md](bonus-templates.md)
  - Bonusförslag för offertsida och modellsidesmall.

## Snabbstart

1. Öppna [index.html](index.html) i webbläsare för att utvärdera den fristående versionen.
2. För live-site på Smartproduktion:
   - Klistra in innehållet i [smartproduktion-patch.html](smartproduktion-patch.html) i custom code på startsidan.
   - Följ QA-checklistan i [IMPLEMENTATION-SMARTPRODUKTION.md](IMPLEMENTATION-SMARTPRODUKTION.md).

## Mål med ändringarna
